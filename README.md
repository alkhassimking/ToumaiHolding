# Toumai Holding | توماي القابضة

Official corporate website for Toumai Holding — a diversified Chadian investment and technology group powering the Toumai digital ecosystem.

## Tech Stack

- **Next.js 15** — React framework with App Router
- **TypeScript** — Type-safe development
- **Tailwind CSS 4** — Utility-first styling
- **Framer Motion** — Smooth animations
- **Three.js / React Three Fiber** — 3D ecosystem background
- **Lucide React** — Icon library

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Features

- Premium black & gold design with glassmorphism
- Full multilingual support (English, French, Arabic + RTL)
- Interactive ecosystem visualization with subsidiary companies
- Animated counters, cards, and scroll effects
- Fully responsive (mobile, tablet, laptop, desktop, ultra-wide)
- SEO optimized with metadata, sitemap, robots.txt, and Schema.org
- Accessibility compliant with skip navigation and ARIA labels
- Dark mode by default
